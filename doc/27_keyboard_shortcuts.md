# 図・表キーボード操作ヘルプ

## 共通
- `Ctrl+Z / Ctrl+Y` : Undo / Redo
- `Ctrl+F` : 検索バーを開く
- `Ctrl+Plus / Ctrl+Minus / Ctrl+0` : ズームイン / ズームアウト / リセット
- `Ctrl+S` : 保存

## 状態遷移表（AG Grid）
- `Enter` : セル編集開始 / 次の行へ確定移動
- `F2` : 再編集開始
- `Ctrl+C / Ctrl+V` : コピー / 貼付
- `Ctrl+Arrow` : 端まで移動（行・列）
- `Space` : 行折畳/展開トグル（階層行）
- `Alt+MouseWheel` : 横スクロール
- `Ctrl+MouseWheel` : ズーム

## 状態遷移図（React Flow）
- `Arrow Keys` : 選択ノードを 1px 移動（`Shift+Arrow` で 10px）
- `Delete` : 選択ノード/エッジ削除
- `Ctrl+D` : 複製
- `Esc` : 選択解除
- `MouseWheel` : ズーム、`Space+Drag` でパン
- `Ctrl+L` : レイアウト再適用（dagre/elkjs 現在選択値）

## シミュレーション/ログビュー
- `Space` : 再生/一時停止トグル
- `Arrow Right / Left` : 次/前ステップへ移動
- `Home / End` : 先頭 / 最終ステップへジャンプ
